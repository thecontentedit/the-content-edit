Preguntas frecuentes
¿Cómo empiezo con el widget?
Sigue la guía en la pestaña Welcome o ve directo a Setup. Solo necesitas duplicar la plantilla de Notion, crear una integración gratuita y conectar tu base de datos. El proceso tarda menos de 5 minutos.
¿Cómo configuro mi base de datos de Notion?
Tu base de datos debe tener estas propiedades con los nombres exactos: Título (texto), Fecha de publicación (fecha), Attachment (archivo). Usuarios Pro también necesitan: Link (texto), Canva Link (URL). Recomendamos usar nuestra plantilla.
¿Por qué no se ven mis imágenes?
Verifica: ✅ ¿La URL es de la base de datos directamente? ✅ ¿Le diste acceso a la integración desde ··· → Connections? ✅ ¿Subiste imágenes al campo Attachment? ✅ ¿Los nombres de las propiedades están escritos igual que en la plantilla?
El widget dice que no puede acceder a mi base de datos
Esto casi siempre pasa porque el link no es el de la base de datos real. Abre tu base de datos en página completa en el navegador y copia la URL desde ahí. Si ves una flechita junto al título, es una vista vinculada — necesitas ir a la base de datos original.
¿Puedo usarlo para varios clientes?
¡Sí! Puedes crear widgets ilimitados, uno por cada cliente. Cada widget se conecta a una base de datos diferente. Puedes reutilizar el mismo Integration Token para todas — solo conéctalo a cada base de datos por separado.
¿Puedo renombrar las columnas de mi base de datos?
No, las propiedades deben mantener sus nombres exactos. Puedes agregar columnas adicionales, pero no renombrar las que usa el widget.
¿Cómo agrego diseños de Canva? (Pro)
En Canva ve a Compartir → Enlace público y copia ese link. Pégalo en la propiedad Canva Link de tu base de datos. El widget lo mostrará directamente sin necesidad de exportar nada.
¿Cómo agrego videos? (Pro)
Sube videos al campo Attachment, o pega links de Dropbox o Google Drive en la propiedad Link. El widget detecta automáticamente que es video por la extensión.
¿Cómo hago upgrade a Pro?
Ve a la sección Account o haz click en cualquier botón de upgrade. Compra usando el mismo email — tus widgets se actualizan automáticamente sin reconfigurar nada.
¿Ofrecen reembolsos?
El plan Free está disponible para que puedas probar antes de comprar. Si tienes algún problema escríbenos a hola@thecontentedit.digital y te ayudamos.
