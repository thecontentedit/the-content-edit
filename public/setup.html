Preguntas frecuentes
¿Cómo empiezo con el widget?
Sigue la guía en la pestaña Welcome o ve directo a Setup. Solo necesitas duplicar la plantilla de Notion, crear una integración gratuita y conectar tu base de datos. El proceso tarda menos de 5 minutos.
¿Cómo configuro mi base de datos de Notion?
Tu base de datos debe tener estas propiedades con los nombres exactos: Título (texto), Fecha de publicación (fecha), Attachment (archivo). Usuarios Pro también necesitan: Imagen desde (select), Link (texto), Canva Link (URL). Recomendamos usar nuestra plantilla para asegurarte de que todo esté correcto.
¿Por qué no se ven mis imágenes?
Verifica lo siguiente: ✅ ¿La URL es de la base de datos directamente (no de una página)? ✅ ¿Le diste acceso a la integración desde ··· → Connections? ✅ ¿Subiste imágenes al campo Attachment? ✅ ¿Los nombres de las propiedades están escritos igual que en la plantilla? Si sigues con problemas, prueba primero con nuestra plantilla y base de datos de ejemplo.
El widget dice que no puede acceder a mi base de datos
Esto casi siempre pasa porque el link que usaste no es el de la base de datos real. Abre tu base de datos en página completa en el navegador y copia la URL desde ahí. Si ves una flechita junto al título de tu base de datos, es una vista vinculada — necesitas ir a la base de datos original.
¿Puedo usarlo para varios clientes?
¡Sí! Puedes crear widgets ilimitados, uno por cada cliente o cuenta. Cada widget se conecta a una base de datos diferente en Notion. Puedes reutilizar el mismo Integration Token para todas — solo asegúrate de conectarlo a cada base de datos por separado.
¿Puedo renombrar las columnas de mi base de datos?
No, las propiedades deben mantener sus nombres exactos. Puedes agregar columnas adicionales, pero no renombrar las que usa el widget.
¿Cómo agrego diseños de Canva? (Pro)
Selecciona "Canva" como Imagen desde en tu base de datos. Luego en Canva ve a Compartir → Embed y copia el link de embed. Pégalo en la propiedad Canva Link. Nota: no es la URL del navegador, sino el link de embed específico.
¿Cómo agrego videos? (Pro)
Puedes subir videos directamente al campo Attachment, o pegar links de Dropbox o Google Drive en la propiedad Link. Selecciona "Link" como Imagen desde. El widget detecta automáticamente que es video por la extensión (.mp4, .mov, .webm).
¿Cómo hago upgrade a Pro?
Ve a la sección Account o haz click en cualquier botón de upgrade. Compra usando el mismo email con el que compraste el Free y tus widgets se actualizan automáticamente — sin necesidad de reconfigurar nada.
¿Ofrecen reembolsos?
El plan Free está disponible para que puedas probar el widget antes de comprar. Si tienes algún problema, escríbenos a hola@thecontentedit.digital y te ayudamos antes de que decidas.
