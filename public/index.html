El problema

Planeas en Notion. Diseñas en Canva. Y para ver tu feed tienes que armar un mockup a mano cada vez.

Cambias una foto y el mockup ya no sirve. Cambias el orden y empiezas de cero. Es un ciclo que no debería existir. The Content Edit conecta tu base de datos de Notion con una vista de grid en tiempo real — actualizas tu base de datos en Notion y tu feed se refleja al instante.

Cómo funciona

Setup de una sola vez. Después es automático.

1
Duplica la plantilla de Notion
Te damos una base de datos lista para usar con todas las propiedades configuradas. O usa la tuya si ya tienes una.
2
Conecta tu integration token
Crea una integración gratuita en Notion, pega tu token en la página de setup y listo. Tarda menos de 3 minutos.
3
Pega tu widget en Notion
Recibes un link único. Lo pegas como bloque /embed en cualquier página de Notion y tu grid aparece al instante.
4
Actualiza tu base de datos en Notion, el grid se actualiza solo
Cada vez que agregas un post, cambias una imagen o reordenas — solo refresca. Tu feed siempre está al día.
